# StringCalculator

## Prerequisites
- jdk 21+

## Running Tests
Linux/Cygwin/WSL/Mac
```bash
./mvnw test
```
Windows
```batch
mvnw test
```

## Start program

To start the program u run main.
inputs can be as followed:
replace number with any number u want
```bash
scalc ‘number,number,number’
```
```bash
number,number,number
```
